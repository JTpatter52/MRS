Allwebco Template
 This template includes three picture galleries, Flash animation and gallery pages with and without PayPal order forms. The Flash logo text displayed above is edited with Notepad or any plain text editor with your website name or company name. All Flash phrases can be edited with Notepad as well. Flash software is not required to edit any areas on this template. The header area is showing a Flash logo. You can easily use your own graphic logo by renaming one text file. View this page with a Flash logo.


Colors and Formatting
 This template includes five color themes. Colors are easily, globally changed for the entire template by editing a plain text file in one place with your choice. Instantly change the color of your website. Template font sizes and colors, page buttons and menu colors and fonts are all set in global CSS files for easy formatting updates. You can also setup your own custom color theme. If you are creating your own theme some graphics editing will be required.


Features
 Plain text global menus, global sidebar areas, header and footers are setup for easy website updates. The drop menus can be setup 3 deep for larger websites. A sample multi-level menu file is included with the download. Or any menu item can be set as a plain button with no drop menu. The sidebar search can be switched off, setup as a Google search or, 3 options are included for a site search. The sidebar scroller is displaying 3 basic HTML pages and is updated like any other template page. The right sidebar can be switched off if not required. View the step by step instructions.
